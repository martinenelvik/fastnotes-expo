# Kjøring (Expo / React Native)

## Forutsetninger
- Node.js + npm installert
- Expo Go installert på mobilen (iPhone/Android)

## Start i utvikling
1. Åpne prosjektmappa i VS Code
2. Installer avhengigheter: npm install 
3. Start utviklingsserver: npx expo start 
4. Test:
Installer Expo Go på mobilen og skann QR-koden som vises i terminalen for å kjøre appen på telefonen 
(Evt. kan du også kjøre på Android-emulator med npm run android.)


OBS! **mobil og PC må være på samme nett**
